# Prism

A app that returns an image in random.

## Quick start.

 - Put some jpg files to `images` folder.

 - `npm start`.

## Config.

`Host` and `Post` config are saved in `index.js`.

## Preset images

All preset images are downloaded from [Pexels](https://www.pexels.com) and CC0-Licensed.
